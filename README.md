
# RAG Faithfulness Study

A mini research project evaluating whether retrieval-augmented prompting improves answer faithfulness and reduces hallucination in question-answering tasks.

## Research Question
Does retrieval-augmented prompting reduce hallucination and improve faithfulness compared with direct prompting in a small-domain QA setting?

## Hypothesis
Providing retrieved supporting context will improve answer quality and reduce unsupported claims.

## Project Goals
- Compare direct prompting vs retrieval-augmented prompting
- Measure hallucination and faithfulness
- Visualize experimental results
- Document findings in a short research-style report

## Methodology
1. Prepare a small dataset of domain questions and reference passages
2. Generate baseline answers without retrieval
3. Generate RAG-style answers with context
4. Evaluate outputs using a structured rubric
5. Analyze and visualize results

## Metrics
- Correctness
- Faithfulness
- Completeness
- Hallucination Rate
